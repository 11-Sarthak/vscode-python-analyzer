## Development

- **Extension code:** `src/extension.ts`
- **Python analysis script:** `pythonFiles/analyze_functions.py`
- **Build:**  
  ```bash
  npm install
  npm run compile
  ```
- **Debug:**  
  Press F5 in VS Code to launch the Extension Development Host.
